# Getting Started

Dokumen ini membantu kamu menjalankan project **Health-Journal-Summarizer** untuk pertama kalinya.

Project ini dirancang untuk membantu mengubah satu topik kesehatan menjadi ringkasan berbasis jurnal ilmiah dan draft artikel populer gaya Medium.

## Who This Project Is For

Project ini cocok untuk:

- pembelajar agentic AI
- penulis konten kesehatan berbasis jurnal
- pembuat catatan riset
- siapa pun yang ingin membangun workflow riset kesehatan yang lebih terstruktur

## What You Need

Sebelum memulai, pastikan kamu memiliki:

- Python 3.10 atau lebih baru
- Git
- editor seperti VS Code
- koneksi internet untuk mencari referensi jurnal
- pemahaman dasar tentang Markdown dan command line, opsional tetapi membantu
